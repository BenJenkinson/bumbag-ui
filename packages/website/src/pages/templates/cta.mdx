# Call-to-action Templates

## Simple

```jsx-live
<Box backgroundColor="white600" paddingX="major-2" paddingY="major-8">
  <Container breakpoint="desktop">
    <Stack>
      <Box lineHeight="1.2" fontWeight="800" fontSize={{ default: '500', mobile: '400' }}>
        Ready to get crackin'?
        <br/>
        <Text color="primary">Start for free today.</Text>
      </Box>
      <Box>
        <Set>
          <Button palette="primary" size="medium">Get started</Button>
          <Button color="primary" size="medium">Learn more</Button>
        </Set>
      </Box>
    </Stack>
  </Container>
</Box>
```

## Simple horizontal

```jsx-live
<Box backgroundColor="white600" paddingX="major-2" paddingY="major-8">
  <Container breakpoint="desktop">
    <Stack orientation="horizontal" verticalAt="desktop">
      <Box lineHeight="1.2" fontWeight="800" fontSize={{ default: '500', mobile: '400' }}>
        Ready to get crackin'?
        <br/>
        <Text color="primary">Start for free today.</Text>
      </Box>
      <Set justifyContent={{ 'min-widescreen': 'flex-end' }}>
        <Button palette="primary" size="medium">Get started</Button>
        <Button color="primary" size="medium">Learn more</Button>
      </Set>
    </Stack>
  </Container>
</Box>
```
