# Variants

```function-live noInline
const theme = {
  Box: {
    variants: {
      dopebox: {
        defaultProps: {
          altitude: '400',
          backgroundColor: 'hotpink',
          fontFamily: '"Comic Sans MS", sans-serif',
          padding: 'major-2',
          _hover: {
            backgroundColor: 'purple',
            color: 'white'
          }
        }
      }
    }
  }
};

function Example() {
  return (
    <Provider theme={theme}>
      <Box variant="dopebox">A box straight from the Fannypack.</Box>
    </Provider>
  )
}

render(Example)
```
