# Drawer

## Import

```jsx
import { Drawer } from 'fannypack';
```

## Usage

```function-live
function Example() {
  const drawer = Drawer.useState();

  return (
    <div>
      <Drawer.Disclosure {...drawer}>Toggle</Drawer.Disclosure>
      <Drawer {...drawer}>
        This is a side overlay
        <Button onClick={drawer.hide}>Close</Button>
      </Drawer>
    </div>
  )
}
```

### Backdrop

You can hide the backdrop with the `hideBackdrop` prop.

```function-live
function Example() {
  const drawer = Drawer.useState();

  return (
    <div>

        This is a side overlay
        <Button onClick={drawer.hide}>Close</Button>
      </Drawer>
    </div>
  )
}
```

### Animation

```function-live
function Example() {
  const drawer = Drawer.useState({ unstable_animated: true });

  return (
    <div>
      <Drawer.Disclosure {...drawer}>Toggle</Drawer.Disclosure>
      <Drawer fade slide {...drawer}>
        This is a side overlay
        <Button onClick={drawer.hide}>Close</Button>
      </Drawer>
    </div>
  )
}
```
