# Styling components

Fannypack comes with a `styled` function which allows you to create components that have styles attached to them.

## Creating a basic styled component

To create a styled component, import the `styled` function and then invoke it with your desired component & template literal string containing your CSS:

```jsx
import { Box, styled } from 'fannypack';

const FancyBox = styled(Box)`
  background-color: pink;
`;

function Example() {
  return (
    <FancyBox>Hello world</FancyBox>
  );
}
```

## Generating styles based from props

You can generate styles based on your props (and changes in props) by supplying a function (`${props => ...}`) inside your CSS:

```jsx
import { Box, css, styled } from 'fannypack';

const FancyBox = styled(Box)`
  background-color: ${props => props.palette};

  ${props => props.type === 'rounded' && css`
    border-radius: 10px;
  `}
`;

function Example() {
  return (
    <FancyBox palette="pink" type="rounded">Hello world</FancyBox>
  );
}
```

> Note: You can use the `css` utility to embed CSS in your CSS.


## Generating styles based from the theme

You can generate styles based off values in your [Fannypack global theme](/theming) by using the `theme` variable stored in the `props` object:

```jsx
import { Box, css, styled } from 'fannypack';

const FancyBox = styled(Box)`
  background-color: ${props => props.theme.palette.primary};
`;

function Example() {
  return (
    <FancyBox>Hello world</FancyBox>
  );
}
```
