# Stack

## Import

```jsx
import { Stack } from 'bumbag-native';
```

## Usage

The `<Stack>` component that assists in creating layouts where it's children are evenly spaced.

```jsx-live
<Stack>
  <Box backgroundColor="primaryTint">
    <Text>First</Text>
  </Box>
  <Box backgroundColor="primaryTint">
    <Text>Second</Text>
  </Box>
  <Box backgroundColor="primaryTint">
    <Text>Third</Text>
  </Box>
</Stack>
```

### Gap

You can specify a gap spacing on your `<Stack>` with the `spacing` prop.

```jsx-live
<Stack spacing="major-2">
  <Box backgroundColor="primaryTint">
    <Text>First</Text>
  </Box>
  <Box backgroundColor="primaryTint">
    <Text>Second</Text>
  </Box>
  <Box backgroundColor="primaryTint">
    <Text>Third</Text>
  </Box>
</Stack>
```
