# Spinner

The `<Spinner>` component can be used to render a visual cue that something is loading.

## Import

```jsx
import { Spinner } from 'bumbag-native'
```

## Usage

```jsx-live
<Spinner />
```

### Sizes

```jsx-live
<Spinner size="small" />
<Spinner />
<Spinner size="medium" />
<Spinner size="large" />
<Spinner size="40px" />
```

### Colors

```jsx-live
<Spinner color="success" />
<Spinner color="danger" />
<Spinner color="warning" />
```
