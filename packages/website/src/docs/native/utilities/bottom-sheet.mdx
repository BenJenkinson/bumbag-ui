# BottomSheet

The `BottomSheet` component is a wrapper around the [react-native-bottom-sheet](https://gorhom.github.io/react-native-bottom-sheet/) library.

## Install

> Note: This component is external from the `bumbag-native` package.

```
yarn add @bumbag-native/bottom-sheet
// or npm
npm install @bumbag-native/bottom-sheet
```

## Import

```js
import { BottomSheet } from "@bumbag-native/bottom-sheet";
```

## Usage

```jsx-live
<BottomSheet />
```
