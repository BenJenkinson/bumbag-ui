# Radio

Checkboxes are used when a user needs to at most one value from a series of options.

## Import

```jsx
import { Radio } from 'bumbag-native'
```

## Usage

```live
###tab=Default,type=jsx-live
<Radio label="Check me" />###

###tab=Ghost,type=jsx-live
<Radio label="Check me" variant="ghost" />###
```

### Controlled

```function-live
function Example() {
  const [checked, setChecked] = React.useState(false);

  return (
    <Radio
      label="Remember me"
      checked={checked}
      onChange={setChecked}
    />
  )
}
```
